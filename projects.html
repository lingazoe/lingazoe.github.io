Here you can find all the group projects I've done in University with the most recent project at the top.

Wine Quality Predictor - Python

The Wine Quality Predictor is a Python project tasked with estimating wine qualities via various machine learning algorithms. Our goal(s) was to determine if
wine quality can be determined by their physicochemical characteristics, and seeing if certain factors contributed to low or high wine qualities. This project
helped me with improving my Python skills, understanding various machine learning algorithms, and exploring real world scenarios involving data analytics.
Fast Fix - iFixit

The iFixit Fast Fix is a collaborative project tasked with developing instructions for repairing, fixing, or replacing an item that would be useful to others.
These instructions are made in the present using photographs and written instructions, helping users to understand the process and apply it themselves.

This project helped me learn how to properly convey technical concepts to a general audience by simplifying complex repair tasks into straightforward steps
people can understand. Additionally, the project enhanced my research, problem solving, time management, and teamwork skills, as this project was done
with another person.
SEO Writing Project - iFixit

The iFixit SEO Writing Project involves the process of creating and optimizing information on an existing device page in iFixit. With this, it helps users with
understanding devices more easily, and having all the information stored in a single hub. This project helped me learn SEO concepts, improve my research skills,
and enhance my writing skills.
Marketplace Simulator - Matlab

The Marketplace Simulator is a Matlab project tasked with simulating a marketplace like Kijiji or Craigslist. Users have the ability to sell, buy, and search for
items via filters, such as price and location based on individual preference. This project helped me with improving my Matlab and Design skills, as creating the
marketplace had multiple components, which involved constant communication with group members so the project would finish on time. It also improved my
time management skills, as I had the iFixit projects to complete alongside it.
